⚙️ One-Time Setup
CONNECT GOOGLE SHEETS — DATA NEVER LOST
STEP 1

Go to script.google.com → New Project → Delete all code → Paste the Apps Script code provided separately

STEP 2

Create a Google Sheet at sheets.google.com → Copy the Sheet ID from the URL → Paste it into the script where it says YOUR_SHEET_ID_HERE

STEP 3

In Apps Script → Click Deploy → New Deployment → Type: Web App → Execute as: Me → Access: Anyone → Deploy → Copy the Web App URL

STEP 4

Paste the Web App URL below and click Connect

You only do this once. After connecting, all your data syncs to Google Sheets automatically on every action.
